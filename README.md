# TripApi

## Project Introduction

This is a web api project imitation `trip.com`, this project uses the following technology stack:

1. Asp.Net Core (Base web framework)
2. Entity Framework (ORM framework)
3. Postgres Driver (Database connector)
4. Auto Mapper (DTO framework)

## Build and Run

If you want to build this project, you can use the following command:

```bash
dotnet build
```



And if you want to run this project, you can use the following command:

```bash
dotnet run
```
